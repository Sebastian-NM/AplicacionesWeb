S02 - Tarea 03 - Propiedades CSS

Nombre de propiedad Función Posibles valores
display Define cómo se muestra un elemento
  • block: ocupa todo el ancho disponible
  • inline: ocupa solo el espacio necesario
  • inline-block: como inline, pero permite definir tamaño
  • flex: usa flexbox para disposición de elementos
  • grid: usa un sistema de cuadrícula
  • none: oculta el elemento
margin Define el espacio exterior alrededor de un elemento
  • Valores en px: tamaño fijo en píxeles
  • %: relativo al tamaño del contenedor
  • auto: ajusta automáticamente
padding Define el espacio interno entre el contenido y el borde
  • Valores en px: tamaño fijo en píxeles
  • %: relativo al contenedor
  • em: relativo al tamaño de fuente
  • auto: ajusta automáticamente
border Define el borde de un elemento
  • none: sin borde
  • solid: línea continua
  • dashed: línea discontinua
  • dotted: puntos
  • double: doble línea
text-align Alinea el texto dentro de un elemento
  • left: alineado a la izquierda
  • right: alineado a la derecha
  • center: centrado
  • justify: justificado
background-color Define el color de fondo de un elemento
  • Colores en nombre: rojo, azul, etc.
  • RGB: combinación de rojo, verde y azul
  • HEX: código hexadecimal
  • HSL: tono, saturación y luminosidad
cursor Define el tipo de cursor al pasar sobre un elemento
  • pointer: mano indicando clic
  • default: cursor por defecto
  • text: cursor para seleccionar texto
  • move: indica que el elemento es movible
  • not-allowed: indica una acción prohibida
